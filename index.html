Portfolio Projects

Culinary Critics Webpage

As the Scrum Master for this group project, I led the development of a full-stack application utilizing React with Vite, Bootstrap, Node.js, MongoDB, AWS, and Trello. Learning Trello from scratch, I successfully implemented it to manage the project timeline, organize tasks, and foster team collaboration. I also contributed by designing webpage mockups and component layouts in PowerPoint, developing a reusable `ReviewCard` component with multiple subcomponents, and applying UI/UX design principles to enhance the user experience. Additionally, I resolved CSS challenges to ensure consistent styling across the application.

GitHub Link:CarrieCanDo/culinary-critics

Oceans4All

Oceans 4 All was a project I created during my time in the Stiegler EdTech Full Stack Web Development apprenticeship. It marked an early milestone in my coding journey, where I designed my first Figma board to plan the layout and structure of the site. The project involved creating a non-profit website concept to promote free parking for beach access. I implemented a responsive, user-friendly interface using Bootstrap and developed web pages with HTML and CSS, focusing on improving user experience and accessibility. Additionally, I created custom contact cards and feature sections to highlight the website concept and functionality.

GitHub Link: CarrieCanDo/UsingBootstrap

School Management System

The School Management System was an individual project where I built a console-based Java application to efficiently manage information for schools, teachers, and students. I designed and implemented `School`, `Teacher`, and `Student` classes, encapsulating attributes and methods to manage teachers and students effectively. By applying object-oriented programming principles, I created modular and reusable code, showcasing core Java concepts such as class relationships and encapsulation. Additionally, I developed a main interface to facilitate user interactions with the system, ensuring a smooth and functional experience. This project deepened my understanding of Java and solidified my programming fundamentals. GitHub Link: CarrieCanDo/SchoolMgmtSystem
