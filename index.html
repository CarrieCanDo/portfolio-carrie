By applying object-oriented programming principles, I created modular and reusable code, showcasing core Java concepts such as class relationships and encapsulation. Additionally, I developed a main interface to facilitate user interactions with the system, ensuring a smooth and functional experience. This project deepened my understanding of Java and solidified my programming fundamentals. GitHub Link: CarrieCanDo/SchoolMgmtSystem
